# Name

Been-to
旅程先の情報集めを目的としたバックパッカー向けの旅行写真を通じた情報提供サイト

# DEMO

https://been-to.work

# Features

投稿での国名・旅程期間・ワクチンの選択
国名・ワクチンを用いた検索

* ruby 2.5.7
* Rails 5.2.4.1
